# Business System Analysis Prompt

请从第一性原理和系统思维角度分析以下公司、产品或商业模式：

[Company / Product Name]

不要讲励志故事，不要讲融资历史，不要讲创始人传奇。

目标是理解：

**这个系统为什么能赚钱，以及未来为什么还能继续赚钱。**

请按照以下结构进行分析：

---

# Executive Summary

用 3-5 句话总结：

* 这个业务本质上在卖什么？
* 用户为什么愿意付钱？
* 最重要的赚钱机制是什么？
* 最大的护城河是什么？

---

# 1. Customer

回答：

* 谁是真正的客户？
* 谁是使用者？
* 谁是付款者？
* 客户有哪些细分群体？
* 哪个群体贡献最多收入？

输出：

* ICP (Ideal Customer Profile)
* Top Customer Segments

---

# 2. Problem

回答：

* 客户最痛的痛点是什么？
* 不解决这个问题会怎样？
* 目前有哪些替代方案？
* 为什么现有方案不够好？

输出：

* Pain Level (1-10)
* Urgency Level (1-10)
* Existing Alternatives

---

# 3. Value

回答：

* 产品创造了什么价值？
* 是帮用户赚钱？
* 省时间？
* 降低风险？
* 提高效率？
* 获得娱乐？
* 获得身份认同？

量化分析：

* 用户一年能获得多少价值？
* 产品大约捕获了多少比例的价值？

输出：

* Core Value Proposition
* Estimated ROI

---

# 4. Distribution

回答：

* 用户如何发现这个产品？
* 获客渠道有哪些？
* 哪个渠道贡献最大？

分析：

* SEO
* Open Source
* Social Media
* Community
* Enterprise Sales
* Partnerships
* Word of Mouth
* Paid Ads

输出：

* Top Acquisition Channels
* CAC (如果可估算)

---

# 5. Trust

回答：

为什么客户相信他们？

分析：

* Brand
* Community
* Case Studies
* Testimonials
* Open Source Reputation
* Security Compliance
* Industry Recognition

输出：

* Trust Mechanisms
* Switching Risk

---

# 6. Monetization

回答：

如何赚钱？

分析：

* Subscription
* Usage-Based
* Enterprise Contract
* Marketplace
* Advertising
* Services

输出：

* Revenue Model
* Pricing Model
* Revenue Drivers

---

# 7. Scale

回答：

增长是否需要增加人力？

分析：

* Software Leverage
* AI Leverage
* Capital Leverage
* Labor Leverage

输出：

* Scalability Score (1-10)
* Bottlenecks

---

# 8. Moat

回答：

为什么竞争对手不能轻易复制？

分析：

* Technology
* Data
* Network Effects
* Ecosystem
* Brand
* Distribution
* Community

输出：

* Top 3 Moats
* Moat Strength (1-10)

---

# 9. Flywheel

画出业务飞轮：

例如：

More Users
→ More Data
→ Better Product
→ More Users

或者：

More Open Source Adoption
→ More Contributors
→ Better Product
→ More Adoption

输出：

* Flywheel Diagram
* Flywheel Strength (1-10)

---

# 10. Leverage

分析业务的核心杠杆：

* Code
* Media
* Capital
* Labor
* Network

回答：

* 哪种杠杆最重要？
* 哪种杠杆决定了收入上限？

输出：

* Primary Leverage
* Secondary Leverage

---

# Business System Diagram

最后画出完整系统：

Customer

Problem
↓
Value
↓
Distribution
↓
Trust
↓
Monetization
↓

并指出：

1. 最关键的单点成功因素
2. 最大风险
3. 如果你是创始人，未来3年最重要的战略动作
4. 这个商业模式最值得学习的地方
5. 给这个商业系统打分（0-100）
