# Output — Production Layer

Output is not the exhaust pipe of the system. Each output type compounds back into a specific layer.

---

## Output Types & Compounding Loops

| Output | Primary compounding target | Secondary effect |
| ------ | ------------------------- | ---------------- |
| **Code** | Career capital (Principal-level trajectory) | Quant system infrastructure |
| **Writing** | Network + reputation → inbound opportunities | Forces synthesis, clarifies thinking |
| **Video** | Audience → leverage for ideas at scale | Feeds writing pipeline |
| **Building** | Product → revenue optionality | Demonstrates system thinking tangibly |
| **1-1 Mentoring** | Forces you to articulate what you know | Network depth, not breadth |

---

## Which Output to Prioritize

Output selection is not static — it depends on current system state:

| System state | Prioritize |
| ------------ | ---------- |
| High career capital, low network | Writing / Mentoring |
| Strong network, low product | Building |
| High expertise, no external signal | Video / Writing (make it legible) |
| Exploration phase (20% rotation) | Code experiments → document findings |
| Low energy period | Writing (lower activation cost than building) |

---

## Output Compounding Back Into Architecture Layers

```
Code ──────────────────► Career System (income, Principal trajectory)
                    └──► Resource System (quant infrastructure)

Writing / Video ────────► Relationship System (network, reputation)
                    └──► Learning System (synthesis = retention)

Building ───────────────► Career System (product optionality)
                    └──► Finance System (revenue diversification)

1-1 Mentoring ──────────► Relationship System (depth)
                    └──► Mind Layer (teaching = highest-order learning)
```

---

## Minimum Viable Output Cadence

One unit of output per week, regardless of energy level. The unit scales with energy:

- **High energy week** → ship a feature, publish an article, record a video
- **Medium energy week** → write a draft, commit working code, have one mentoring conversation
- **Low energy week** → write one paragraph, push one commit, send one thoughtful message

The cadence matters more than the size. Compounding requires continuity.
