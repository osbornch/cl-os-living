✔ 方法论：可操作但不稳定

— 会因场景不同而失效
— 通常适用于特定条件
— 多由经验总结
— 是“怎么做”

✔ 原理：稳定、不变的底层规律

— 不随场景变化
— 可以解释为什么方法有效
— 是“为什么这样”

✔ 智慧：动态选择正确原理与正确方法

— 能看到长期
— 能做权衡取舍
— 会因人性、系统、时机调整
— 是“何时用、用多少、在谁身上用”
