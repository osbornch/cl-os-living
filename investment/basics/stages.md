# 阶段一：零售与服务（极简单细胞模型）
代表模式： 街角咖啡店、理发店、独立书店。

商业核心： 一手交钱，一手交货。 * 三表特点： * 资产负债表： 极其干净。几乎没有应收账款（没人能赊账喝咖啡），资产里只有现金、少量存货（豆子、杯子）和固定资产（咖啡机）。

利润表： 收入减去原材料成本和电费人工，就是纯利。

现金流量表： 净利润和经营现金流几乎是 1:1 像素级对齐的。赚到一笔账面利润，银行账户里就结结实实多一笔现金。

# 阶段二：渠道与周转（引入“时间差”和“供应链”）
代表模式： 沃尔玛、便利店、网店、手机经销商。

商业核心： 低进高出，利用规模和时间差赚钱。

三表新要素： 引入了“账期（应收/应付）”和“压货（存货）”。

三表进化：

资产负债表： 出现了大量的“存货”、欠供应商的“应付账款”、和欠自己钱的“应收账款”。

利润表与现金流的分化： 这时候两张表开始对不上了。如果货卖出去了但客户没给钱（应收账款），利润表很好看，但现金流会失血。

核心观察指标： 存货周转率（货在仓库里躺多久能卖掉）和现金循环周期。

# 阶段三：工业与制造（引入“重资产”与“产能”）
代表模式： 汽车厂、芯片代工厂、水泥厂、航空公司。

商业核心： 通过巨大的资本投入，建立生产线，大规模复制产品。

三表新要素： 引入了“固定资产”、“在建工程”和“折旧损耗”。

三表进化：

资产负债表： 左边最重的不再是现金和存货，而是连绵的厂房和价值连城的机器设备（固定资产）。

利润表： 多了一只看不见的吞噬利润的手——折旧。哪怕今年一辆车都没卖出去，几条生产线的折旧费也会按时在利润表上扣除。

核心观察指标： 资产负债率（借了多少钱建厂）和固定资产周转率。

阶段四：平台与生态（现代数字经济）
代表模式： 苹果、微软、腾讯、SaaS软件公司。

商业核心： 轻资产运营，赢家通吃，网络效应。

三表新要素： 引入了“合同负债（预收款）”、“无形资产”和“长期股权投资（资本运作）”。

三表进化：

资产负债表： 走向另一个极端。工厂都在别人家（苹果），自己账上全是现金、专利（无形资产）以及投资其他公司的股权（长期股权投资）。右边则堆满了用户提前充值的钱（合同负债）。

利润表： 拥有恐怖的高毛利率（软件复制成本几乎为零）。
